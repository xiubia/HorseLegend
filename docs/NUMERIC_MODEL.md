# 数值模型文档

## 一、速度系统

### 1.1 基础速度公式

```
基础最大速度 = 5 + speedLevel × 0.5  （米/秒）
```

- `speedLevel` 从 0 开始，通过跑步机训练增长
- 训练速度取决于跑步机等级（见训练系统）

### 1.2 坐骑加成后的速度

```
实际最大速度 = 基础最大速度 × (1 + 坐骑极速加成 / 100)
实际加速度   = 10 × (1 + 坐骑加速度加成 / 100)  （米/秒²）
```

### 1.3 AI 跑酷速度

AI 跑酷模式使用独立速度系统，不读取 `speedLevel`：

```
初始速度等级 = PARKOUR_INITIAL_SPEED_LEVEL（固定值）
速度上限 = 35 m/s（受坐骑加成影响后可更高）
加速倍率 = 随距离逐渐增加
```

卡牌效果中的 `speedMultiplier` 会直接乘以目标速度。

## 三、训练系统（跑步机）

### 3.1 跑步机配置

| 等级 | 解锁价(🏆) | 速度增益(/秒) |
|------|-----------|--------------|
| Lv1 | 0（默认） | 1.0 |
| Lv2 | 50 | 2.0 |
| Lv3 | 200 | 3.5 |
| Lv4 | 500 | 5.0 |
| Lv5 | 1000 | 8.0 |

### 3.2 训练公式

```
speedLevel 增量 = 跑步机增益 × deltaTime（每帧累加）
```

玩家站在跑步机上即自动训练，离开即停止。`speedLevel` 无上限。

## 四、收集物与经济

### 4.1 收集物类型

| 类型 | ASCII | 基础价值 | 说明 |
|------|-------|---------|------|
| 金币 | C | 1 | 常见收集物 |
| 星星 | S | 5 | 稀有高价值 |

### 4.2 实际收集价值

```
实际价值 = 基础价值 × 卡牌金币倍率(coinMultiplier) × (1 + 坐骑金币加成 / 100)
```

### 4.3 奖杯奖励（AI 跑酷）

```
奖杯 = floor(距离 / 20) + floor(收集分 / 2)
```

### 4.4 奖杯奖励（竞速模式）

由 `RewardSystem` 根据距离和排名计算，具体公式见 `RewardSystem.ts`。

## 五、Roguelike 卡牌效果

### 5.1 效果参数

| 效果 | 字段 | 默认值 | 范围 | 说明 |
|------|------|--------|------|------|
| 速度倍率 | speedMultiplier | 1.0 | 0.5~2.0 | 乘以当前目标速度 |
| 赛道宽度 | trackWidth | 6 | 4~10 | 赛道横向宽度（米） |
| 金币倍率 | coinMultiplier | 1 | 1~5 | 金币价值翻倍 |
| 护盾 | shieldCount | 0 | 0~3 | 吸收碰撞次数 |
| 复活 | reviveCount | 0 | 0~1 | 死亡后复活次数 |
| 反转操控 | invertControls | false | - | 左右方向互换 |
| 迷雾 | fogLevel | 0 | 0~3 | 增加雾浓度 |
| 磁铁 | magnetActive | false | - | 自动吸取附近金币 |

### 5.2 效果持续时间

- 所有临时效果（速度/迷雾/反转/磁铁）有时间限制，到期自动消除
- 护盾和复活不受时间限制，按次数消耗
- 赛道宽度变化在下一个 chunk 生效

### 5.3 负面卡牌补偿

负面效果的卡牌搭配正面奖励：
- 迷雾 → 附赠金币翻倍
- 反转操控 → 附赠护盾
- 赛道变窄 → 附赠速度加成

## 六、碰撞检测

### 6.1 跑酷碰撞

```
碰撞判定 = |playerX - obstacleX| < (obstacleWidth + playerWidth) / 2
         && |playerZ - obstacleZ| < 1.2
```

- 只检测 `isObstacle: true` 的资产
- 已隐藏的障碍（被护盾消耗后）跳过检测
- 装饰物（树/花/灌木/路灯/门）不触发碰撞

### 6.2 擦肩而过（Near Miss）

```
擦肩判定 = dx > collisionWidth && dx < collisionWidth × 1.5 && dz < 1.2
```

触发红色闪烁特效 + AI 弹幕评论。

### 6.3 护盾消耗

碰撞时若 `shieldCount > 0`：
1. `shieldCount--`
2. 障碍物 `mesh.visible = false`（防止重复碰撞）
3. 显示护盾吸收提示
4. 不触发死亡

## 七、难度曲线

### 7.1 AI 跑酷难度进阶

| chunk 编号 | 最大障碍数 | 最小行间距 | 说明 |
|-----------|-----------|-----------|------|
| 0-2 | 0 | - | 固定入门段，无障碍 |
| 3-5 | 3 | 4 行 | AI 温和生成 |
| 6-10 | 5 | 3 行 | 正常难度 |
| 11+ | 7 | 2 行 | 高难度 |

### 7.2 速度与障碍密度联动

```
高速修正 minRowGap = max(基础 minRowGap, floor(当前速度 / 8))
```

速度越快，障碍行间距越大，确保玩家有足够反应时间。

### 9.3 加成计算公式

训练速度：

```
实际训练增益 = 基础增益
             × (1 + 天赋训练加成 / 100)
             × 重生训练倍率
             × (1 + 训练师训练加成总计 / 100)
             × 双倍概率判定
```

奖杯/金币收益：

```
实际奖杯 = 基础奖杯
          × (1 + 天赋奖杯加成 / 100)
          × (1 + 训练师金币加成总计 / 100)

跑酷收集物价值 = 基础价值
               × 卡牌金币倍率
               × (1 + 坐骑金币加成 / 100)
               × (1 + 天赋金币加成 / 100)
               × (1 + 训练师金币加成总计 / 100)
```

## 十、宠物系统

### 10.1 宠物概述

宠物为**抽蛋型**系统：消耗奖杯进行抽取，获得的宠物存入背包，装备后跟随马匹并提供速度加成。

- 最大持有数：50 只
- 最大装备数：3 只（装备的宠物加成叠加）
- 重复宠物可获得

### 10.5 加成计算

宠物速度加成 = `(倍率 - 1) × 10`%

| 倍率 | 速度加成 |
|------|---------|
| 1.2x | +2% |
| 1.6x | +6% |
| 2.4x | +14% |
| 4.0x | +30% |

装备多只宠物时，加成叠加：

```
实际最大速度 = 基础最大速度
             × (1 + 坐骑极速加成 / 100)
             × (1 + 天赋移速加成 / 100)
             × (1 + 宠物速度加成总计 / 100)
```
