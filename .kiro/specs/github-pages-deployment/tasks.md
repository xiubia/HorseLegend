# 实施计划：GitHub Pages 部署功能

## 概述

本实施计划将为 HorseLegend 3D 游戏项目添加 GitHub Pages 部署能力。主要工作包括：配置 Vite 构建系统支持子路径部署、建立 GitHub Actions 自动化流程、确保资源路径正确处理、提供本地预览能力，以及编写部署文档。

    - 配置构建输出目录为 dist
    - 启用 source map 生成
    - 配置 Terser 代码压缩
    - 配置资源文件命名规则（包含内容哈希）
    - 配置代码分割策略（Three.js 单独打包）
